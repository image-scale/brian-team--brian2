# Goal

## Project
brian2 — a python project.

## Description
A clock-driven simulator for spiking neural networks. Brian makes it easy to define neural models using differential equations and simulate them efficiently. Users can create groups of neurons with equations defining their dynamics, connect them with synapses, and record their activity during simulation.

## Scope
- ~30 production source files to implement
- ~15 test files to write
- Reproduce core functionality: units system, clocks, network simulation, neuron groups, synapses, monitors, and input generators
